# 配色模特试色预览 · 豆包提示词临时文档

> 创建日期：2026-05-13
> 状态：临时讨论稿
> 用途：沉淀当前用于豆包实验的提示词，便于后续继续讨论、追修和筛选。
> 说明：本文档只用于记录提示词，不替代当前主路线文档 [`palette-outfit-preview-asset-generation-guide.md`](./palette-outfit-preview-asset-generation-guide.md)。

## 1. 使用原则

当前这组提示词的目标不是“直接生成最终可交付 SVG”，而是先拿到更稳定的参考母版与服装结构图，用来判断：

1. 人物比例和站姿是否稳定。
2. 服装结构是否足够清楚。
3. 后续是否有机会整理成 `main / secondary / accent` 三个色区。

建议第一轮只生成四类图：

1. 母版 croquis
2. 短袖上衣
3. 长裤
4. 中长连衣裙

### 1.1 首轮结果判断

1. 第一张图的优势是站姿自然，头身比、肩颈线、手脚位置都更接近可继续复用的人物母版，且面部保留了轻微五官。
2. 第一张图的问题是体积感和柔和阴影偏多，更像插画成品，不够接近后续可整理为 SVG 的纯色块面。
3. 第二张图的优势是纯色矢量块面方向更接近目标，服装和人物外轮廓更容易继续抽象成模板。
4. 第二张图的问题是正面站姿偏僵，人物气质过于简化，面部被压缩成“无五官”状态，不利于保留编辑感和东方女性审美特征。
5. 第二轮目标应明确收口为：保留第一张图的人体姿态和轻微五官，同时把画面压回第二张图那种更接近纯色矢量模板的平面表达。

### 3.5 第二轮融合版提示词

这一组提示词专门解决首轮结果里的核心矛盾：

1. 姿态和人物气质尽量靠近第一张图。
2. 平面纯色和可继续抽象为 SVG 的方向尽量靠近第二张图。

#### 3.5.1 双参考图控制句

如果豆包允许同时上传两张参考图，建议这样分工：

1. 参考图 1：使用第一张“姿态更自然、带轻微五官”的人物图。
2. 参考图 2：使用第二张“更接近平面矢量块面”的短袖图。

先加这句总控：

```text
请综合两张参考图：优先保留参考图 1 的人物站姿、头身比、肩颈线、手脚位置和轻微五官，同时把最终画面风格控制为接近参考图 2 的纯色平面矢量效果。不要做成带明显体积光影的插画成品，也不要做成无五官的极简人形图标。
```

#### 3.5.4 如果只能上传一张参考图

如果豆包一次只能用一张参考图，优先上传第一张姿态更好的图，然后在提示词里额外补这句：

```text
请保持这张参考图的人体站姿和人物气质，但把最终画面压成更接近纯色矢量块面的平面表达：减少体积阴影，减少皮肤和衣料渐变，保留清晰外轮廓和少量必要结构线，便于后续转成 SVG。
```

## 6. 跑偏时的追修句

如果第一轮结果跑偏，优先追加下面这些追修句，不要立刻重写整条提示词：

### 6.1 去场景

```text
把场景全部去掉，只保留单人全身和纯白背景。
```

### 6.2 降低写实感

```text
改成更接近可编辑矢量稿的平面块面风格，减少光影和材质细节。
```

### 6.3 强化服装可见性

```text
双臂与躯干之间留出明显空隙，不要遮挡上衣主体结构。
```

### 6.4 强化三分区结构

```text
服装结构再简化一些，强调可拆分的主体区、次级区、点缀区。
```

### 6.5 锁定同一人物

```text
保持同一人物比例和站姿，不要换角色，不要换构图。
```

### 6.6 恢复轻微五官

```text
保留简洁但可辨识的五官，不要无脸；眉眼鼻唇都要有轻微表达，但不要浓妆，不要过度写实。
```

### 6.7 强化中国女性气质

```text
人物气质更接近现代中国女性的审美特征，清爽、克制、温和，不要欧美浓妆感，也不要日漫夸张五官。
```

## 7. 第一轮建议执行顺序

建议按下面顺序生成：

1. 先出 `croquis` 母版。
2. 再出 `short-sleeve`。
3. 再出 `trousers`。
4. 最后出 `midi-dress`。

如果第一张母版的人体比例和站姿不稳定，后面三张不要继续细跑，先把母版收敛到可复用状态。

## 8. 当前判断标准

第一轮结果只看下面四件事：

1. 人物是否是稳定的单人女性全身站姿。
2. 服装结构是否足够简洁、可拆分。
3. 三分区是否能看出来，不需要后续大幅重画。
4. 四张图之间是否接近同一人物与同一构图。

如果这四项里有两项以上明显不满足，就不建议继续在豆包里深挖，直接回到矢量资产 shortlist 主路线。
